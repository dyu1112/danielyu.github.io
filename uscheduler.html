USCheduler

Role(s): Backend Developer
Technologies Used: Java, Java Servlets, MySQL Workbench
Feb. 2023 - Apr. 2023



Responsibilities:

  • Created 4 Java Servlets on the backend to allow for access to the MySQL database
  • Created input validation for registering, logging in, and creating events
  • Implemented functionality to a JDBC connector class to retrieve all events on a specified date
  • Created weekly documentation under the Waterfall design methodology

Created for CSCI-201: Principles of Software Development. USCheduler is a web application using Java Servlets on the backend with Apache Tomcat. USCheduler allows guests and authenticated users to view scheduled fundraisers on a public calendar. Authenticated users can add their own fundraisers to the public calendar for everyone to see.
